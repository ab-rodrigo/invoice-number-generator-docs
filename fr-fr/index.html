Bienvenue

Apprenez à installer, configurer et créer des préfixes dynamiques pour vos factures.

Aperçu Général

Créez des préfixes dynamiques et des numéros de facture uniques et standardisés pour chaque boutique.

La solution ultime pour les commerçants souhaitant standardiser et organiser leur numérotation de factures. Grâce à une approche innovante basée sur des préfixes dynamiques, ce module vous permet de créer des règles personnalisées pour chaque boutique.

Prérequis

  • Installateur et Gestionnaire d'extensions
  • Gestionnaire de commandes

Structure du Numéro de Facture

La formation du numéro de facture est modulaire et flexible, divisée en deux composantes.

PréfixeABC-XYZ-
Séquentiel0001

Préfixes

Identifiants macro qui précèdent le numéro séquentiel (ex : ABC-XYZ-).

  • Modulaire : Segmenté en plusieurs blocs.
  • Évolutif : Ajoutez autant de blocs que vous le souhaitez.

Séquence Numérique

Le cœur séquentiel obligatoire (ex : 0001), garantissant l'unicité.

  • Padding : Remplissage de zéros à gauche.

Attention : Sensibilité aux Séparateurs

La structure traite chaque caractère littéralement, liant la séquence numérique à la combinaison de préfixes et de séparateurs. Tout changement — comme le remplacement d'un trait d'union (-) par une barre oblique (/) — définit une nouvelle identité, réinitialisant automatiquement la séquence numérique pour cet identifiant spécifique.

ABC-XYZ-0001 Modèle de référence.
ABC/XYZ-0001 La barre modifie le préfixe : le comptage redémarre pour ce nouveau groupe.

Conseil de Standardisation :

Pour maintenir la lisibilité sur les étiquettes et les rapports, utilisez des acronymes courts pour la représentation.

Recommandé : INV-ST1-001 (Invoice - Store 1)
Évitez : INVOICE-STORE01-001

Installation

Suivez les étapes ci-dessous pour intégrer le module en toute sécurité à votre système OpenCart.

01

Téléchargement

Obtenez le module officiel directement sur la marketplace OpenCart :

Page Officielle
02

Upload

Dans le panneau d'administration, allez dans Extensions > Installateur. Cliquez sur Upload et sélectionnez le fichier téléchargé.

03

Activation

Localisez le module dans la liste des extensions installées et cliquez sur l'icône Installer pour l'activer.

Conseil technique : Après l'activation, n'oubliez pas d'accéder à Extensions > Modifications et de cliquer sur le bouton Actualiser (icône bleue) pour vider le cache du système.

Accès aux Paramètres

Après l'installation, suivez ce flux pour configurer votre automatisation.

01

Menu Latéral

Accédez à Extensions > Extensions.

02

Filtre

Sélectionnez le type Modules.

03

Édition

Cliquez sur Modifier pour ouvrir le panneau.

1. Configuration Générale

ParamètreFonction
Génération AutomatiqueCrée le numéro de facture lors de la confirmation de la commande.
LongueurDéfinit le nombre de chiffres du numéro séquentiel (Remplissage de zéros à gauche).
StatutActive le module.

2. Préfixe

Cet onglet permet de composer les éléments de texte ou de date.

ParamètreDescription
Type Définit si l'élément sera un Texte Fixe ou une Date Dynamique.
Contenu (Texte) La valeur textuelle qui sera affichée (ex: INV).
Format (Date) Le modèle de date souhaité (ex: Année deux chiffres + mois).
Séparateur Interne Caractère qui unira cet élément au suivant.

Astuce : Vous pouvez ajouter plusieurs éléments pour créer des préfixes complexes, comme INV-2026-ST01-.

Générer des numéros de facture

Il existe deux façons de générer les numéros de facture :

1. Automatisé

Lorsque la Génération Automatique (onglet Configuration Générale) est activée, le numéro de facture est généré au moment de la confirmation de la commande, éliminant ainsi toute action manuelle.

2. Manuel

Si l'option est désactivée, le numéro de facture suit le flux standard d'OpenCart. La génération doit être effectuée manuellement en suivant ces étapes :

01

Ventes

Dans le menu latéral, accédez à Ventes > Commandes.

02

Détails

Localisez la commande souhaitée et cliquez sur le bouton Afficher.

03

Générer la facture

Cliquez sur Générer à côté du champ Facture.

Désinstallation

Suivez les étapes ci-dessous pour une désinstallation propre et sécurisée.

01

Désactiver

Allez dans Extensions > Extensions, filtrez par Modules, localisez le module et cliquez sur Désinstaller.

02

Désinstaller

Localisez le module dans la liste Extensions > Installateur ou dans la liste des extensions et cliquez sur l'icône Désinstaller.

03

Supprimer

Toujours dans la liste des extensions installées, localisez le fichier et cliquez sur Supprimer.

Qu'advient-il des données ?

La désinstallation supprime les paramètres et les fichiers du module. Cependant, les numéros générés resteront sauvegardés dans la base de données pour éviter toute perte d'intégrité de vos dossiers.
