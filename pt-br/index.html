Bem-vindo

Aprenda a instalar, configurar e criar prefixos dinâmicos para suas faturas.

Visão Geral

Crie prefixos dinâmicos e números de faturas exclusivos e padronizados para cada loja.

Solução definitiva para lojistas que buscam padronizar e organizar seus números de faturas. Com uma abordagem inovadora baseada em prefixos dinâmicos, este módulo permite criar regras personalizadas para cada loja.

Requisitos

Certifique-se de ter permissões em:

  • Instalador e Gerenciador de Extensões
  • Gerenciador de pedidos

Estrutura do Número da Fatura

A formação do número da fatura é modular e flexível, dividida em dois componentes.

PrefixoABC-XYZ-
Sequencial0001

Prefixos

Identificadores macro que precedem o número sequencial (ex: ABC-XYZ-).

  • Modular: Segmentado em múltiplos blocos.
  • Escalável: Adicione quantos blocos desejar.

Sequencial Numérico

O núcleo sequencial obrigatório (ex: 0001), garantindo a exclusividade.

  • Padding: Preenchimento de zeros à esquerda.

Atenção: Sensibilidade aos Separadores

A estrutura processa cada caractere de forma literal, vinculando a sequência numérica à combinação de prefixos e separadores. Qualquer alteração — como a troca de um hífen (-) por uma barra (/) — define uma nova identidade, reiniciando automaticamente a sequência numérica para esse identificador específico.

ABC-XYZ-0001 Padrão de referência.
ABC/XYZ-0001 A barra altera o prefixo: contagem reinicia para este novo grupo.

Dica de Padronização:

Para manter a legibilidade em etiquetas e relatórios, utilize siglas curtas para representar.

Recomendado: INV-ST1-001 (Invoice - Store 1)
Evite: INVOICE-STORE01-001

Instalação

Siga as etapas abaixo para integrar o módulo ao seu sistema OpenCart de forma segura.

01

Download

Obtenha o módulo oficial diretamente no marketplace do OpenCart:

Página Oficial
02

Upload

No painel da loja, acesse Extensões > Instalador. Clique em Upload e selecione o arquivo baixado.

03

Ativação

Localize o módulo na lista de extensões instaladas e clique no ícone de Instalar para ativá-lo.

Dica Técnica: Após a ativação, lembre-se de acessar Extensões > Modificações e clicar no botão Atualizar (ícone azul) para limpar o cache do sistema.

Acesso às Configurações

Após a instalação, siga este fluxo para configurar sua automação.

01

Menu Lateral

Acesse Extensões > Extensões.

02

Filtro

Selecione o tipo Módulos.

03

Edição

Clique em Editar para abrir o painel.

1. Configuração Geral

ParâmetroFunção
Preenchimento AutomáticoCria o número da fatura ao confirmar o pedido.
ComprimentoDefine o número de dígitos do número sequencial (Preenchimento de zeros à esquerda).
StatusHabilita o módulo.

2. Prefixo

Esta aba permite compor os elementos de texto ou data.

ParâmetroDescrição
Tipo Define se o elemento será um Texto Fixo ou uma Data Dinâmica.
Conteúdo (Texto) O valor textual que será exibido (ex: INV).
Formato (Data) O padrão de data desejado (ex: Ano dois dígitos + mês).
Separador Interno Caractere que unirá este elemento ao próximo.

Dica: Você pode adicionar múltiplos elementos para criar prefixos complexos, como INV-2026-ST01-.

Gerar Números de Fatura

Existem duas maneiras de gerar os números de fatura:

1. Automatizado

Com o Preenchimento Automático (aba Configuração Geral) ativado, o número da fatura é gerado no momento da confirmação do pedido, eliminando a necessidade de qualquer ação manual.

2. Manual

Se a opção estiver desativada, o número da fatura segue o fluxo padrão do OpenCart. A geração deve ser feita manualmente seguindo estas etapas:

01

Vendas

No menu lateral, acesse Vendas > Pedidos.

02

Detalhes

Localize o pedido desejado e clique no botão Exibir.

03

Gerar Fatura

Clique em Gerar ao lado do campo Fatura.

Desinstalação

Siga os passos abaixo para uma desinstalação limpa e segura.

01

Desativar

Vá em Extensões > Extensões, filtre por Módulos, localize o módulo e clique em Desinstalar.

02

Desinstalar

Localize o módulo na lista de Extensões > Instalador ou na lista de extensões e clique no ícone Desinstalar.

03

Excluir

Ainda na lista de extensões instaladas, localize o arquivo e clique em Excluir.

O que acontece com os dados?

A desinstalação remove as configurações e arquivos do módulo. No entanto, os números gerados permanecerão salvos no banco de dados para evitar perda de integridade nos seus cadastros.
