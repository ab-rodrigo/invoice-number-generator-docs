### Transform your store's invoice management
The **Invoice Number Generator** eliminates manual invoice creation and brings organizational compliance to your e-commerce. Unlike OpenCart's limited default settings, this module offers total freedom to create professional and dynamic invoice numbers.

👉 **Download the extension on the OpenCart Marketplace:**
https://www.opencart.com/index.php?route=marketplace/extension&filter_member=Rodrigoabr

---

### Key Features:

| Feature | Benefit for the Merchant |
| :--- | :--- |
| **🚀 Full Automation** | The invoice number is generated upon order confirmation. No extra clicks. |
| **🏢 Multi-store Power** | Distinct prefixes for each store. Real organization. |
| **📅 Dynamic Prefixes** | Use date/year tags for invoices like `INV-2026-001`. |
| **🌍 Global** | Interface in PT, EN, ES, FR, and IT. Perfect for international agencies. |
