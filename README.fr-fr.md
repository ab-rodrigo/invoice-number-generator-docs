### Transformez la gestion des factures de votre boutique
L'**Invoice Number Generator** élimine la génération manuelle des factures et apporte une conformité organisationnelle à votre e-commerce. Contrairement au standard limité d'OpenCart, ce module offre une liberté totale pour créer des numéros de facture professionnels et dynamiques.

👉 **Téléchargez l'extension sur l'OpenCart Marketplace :**
https://www.opencart.com/index.php?route=marketplace/extension&filter_member=Rodrigoabr

---

### Caractéristiques Principales :

| Caractéristique | Bénéfice pour le Marchand |
| :--- | :--- |
| **🚀 Automatisation Totale** | Le numéro de facture est généré à la confirmation de la commande. Sans clics supplémentaires. |
| **🏢 Puissance Multi-boutique** | Des préfixes distincts pour chaque boutique. Une organisation réelle. |
| **📅 Préfixes Dynamiques** | Utilisez des balises de date/année pour des factures comme `INV-2026-001`. |
| **🌍 Global** | Interface en PT, EN, ES, FR et IT. Parfait pour les agences internationales. |
