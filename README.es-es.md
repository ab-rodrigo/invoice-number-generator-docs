### Transforme la gestión de facturas de su tienda
El **Invoice Number Generator** elimina la generación manual de facturas y aporta cumplimiento organizacional a su comercio electrónico. A diferencia del estándar limitado de OpenCart, este módulo ofrece total libertad para crear números de factura profesionales y dinámicos.

👉 **Descargue la extensión en el OpenCart Marketplace:**
https://www.opencart.com/index.php?route=marketplace/extension&filter_member=Rodrigoabr

---

### Características Principales:

| Característica | Beneficio para el Comerciante |
| :--- | :--- |
| **🚀 Automatización Total** | El número de factura se genera al confirmar el pedido. Sin clics adicionales. |
| **🏢 Poder Multitienda** | Prefijos distintos para cada tienda. Organización real. |
| **📅 Prefijos Dinámicos** | Use etiquetas de fecha/año para facturas como `INV-2026-001`. |
| **🌍 Global** | Interfaz en PT, EN, ES, FR e IT. Perfecto para agencias internacionales. |
