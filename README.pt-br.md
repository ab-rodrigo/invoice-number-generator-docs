### Transforme a gestão de faturas da sua loja
O **Invoice Number Generator** elimina a geração manual de faturas e traz conformidade organizacional para o seu e-commerce. Diferente do padrão limitado do OpenCart, este módulo oferece total liberdade para criar números de faturas profissionais e dinâmicos.

👉 **Baixe a extensão no OpenCart Marketplace:**
https://www.opencart.com/index.php?route=marketplace/extension&filter_member=Rodrigoabr

---

### Principais Recursos:

| Recurso | Benefício para o Lojista |
| :--- | :--- |
| **🚀 Automação Total** | O número da fatura é gerado o pedido. Sem cliques extras. |
| **🏢 Poder Multiloja** | Prefixos distintos para cada loja. Organização real. |
| **📅 Prefixos Dinâmicos** | Use tags de data/ano para faturas como `INV-2026-001`. |
| **🌍 Global** | Interface em PT, EN, ES, FR e IT. Perfeito para agências internacionais. |
